# data/

Derived datasets produced by the scripts in [`../scripts/`](../scripts/).

## Committed files (small, structural)

| file | description |
|---|---|
| `ratio_top20.csv` | Top 20 ratios $T_{k+1}/T_k$ from the $10^9$ twin set. |
| `ratio_top20_1e10.csv` | Top 20 ratios from the $10^{10}$ twin set. |
| `near_misses_1e10.csv` | All pairs with $r_k > 1.1$ and $T_k \ge 11$ in the $10^{10}$ range. |
| `decade_gap_table.csv` | Per-decade twin-gap statistics (to $10^9$). |
| `decade_table_1e10.csv` | Per-decade twin-gap statistics (to $10^{10}$). |
| `envelope_fits.csv` | Per-constellation envelope fits $(C, \delta, R^2)$ and pooled uniform fit. |
| `constellation_ratios.csv` | Comparative summary: twin, cousin, sexy Bertrand-bound statistics. |
| `angle_records_1e8.csv` | 440,312 angle-record pairs for $p_n < 10^8$ (~20 MB). |
| `twin_ramanujan_primes.csv` | First $10^6$ terms of $R^{\mathrm{twin}}_n$ (~19 MB). |

## Gitignored (large, regenerable)

The following binary arrays are not committed. Each regenerates from one of the scripts in minutes.

| file | size | regenerated by |
|---|---|---|
| `twins_1e9.npy` | 27 MB | `../scripts/pg_twin_angle_analysis.py` (~30 s) |
| `twins_1e10.npy` | 219 MB | `../scripts/pg_extend_1e10.py` (~5 min) |
| `cousins_1e10.npy` | 219 MB | `../scripts/pg_structural_1e10.py` (~6 min, also produces sexy) |
| `sexy_1e10.npy` | 439 MB | `../scripts/pg_structural_1e10.py` |

Format: each `.npy` is a NumPy `int64` array of the smaller members of each pair (i.e., for twins, the $T_k$; for cousins, the $p$ in $(p, p+4)$; for sexy, the $p$ in $(p, p+6)$).

## Reproducing everything from scratch

```bash
cd scripts/
python pg_twin_angle_analysis.py        # → twins_1e9.npy, angle_records_1e8.csv, small tables
python pg_extend_1e10.py                # → twins_1e10.npy, decade_table_1e10.csv, ratio_top20_1e10.csv
python pg_structural_1e10.py            # → cousins_1e10.npy, sexy_1e10.npy, constellation_ratios.csv
python pg_envelope_fits.py              # → envelope_fits.csv
python pg_twin_ramanujan.py             # → twin_ramanujan_primes.csv
```

Total wall time: ~15 minutes on a modern laptop.
